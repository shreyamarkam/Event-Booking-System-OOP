# Event Booking System (OOP)

## 📌 Description
A web-based Event Booking System developed using C++ and Object-Oriented Programming concepts. The system allows users to view events, book tickets, manage bookings, and perform transactions.

## 🛠 Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Basic Web Interface (HTML/CSS)

## 🤝 Contribution
This project was based on an existing implementation. I contributed by:
- Implementing a Payment Gateway feature for booking transactions
- Adding functionality to Cancel Bookings
- Developing feature to Add New Events dynamically
- Improving overall system functionality and user interaction

## 🚀 Features
- View available events
- Book tickets
- Cancel bookings
- Add new events
- Payment gateway integration

## 💡 OOP Concepts Used
- Classes & Objects
- Inheritance
- Polymorphism
- Encapsulation
